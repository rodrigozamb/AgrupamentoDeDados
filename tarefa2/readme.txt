Intruções para execução do programa em Python referente a tarefa 2 da disciplina de Agrupamento de Dados:
    . O Data set escolhido foi o "rating_final.csv" ( https://archive.ics.uci.edu/ml/datasets/Restaurant+%26+consumer+data ), que representa avaliações de diversos clientes a respeito de diversos restaurantes. Os colunas devem ser indexadas a partir de 0, e como as duas primeiras colunas representam Ids elas não são elegíveis para substituição de valores ausentes. Sendo assim as colunas válidas são as colunas 2,3 e 4.
    . Existem 3 arquivos essenciais para execução do oque foi pedido no enunciado da tarefa e para validação da resposta, são eles:
        - tarefa2.py = Arquivo que irá fazer a leitura do database de entrada ira fazer a estimação dos valores ausentes e irá gerar um arquivo "result.csv" com os dados preenchidos. O programa exige o nome do arquivo csv de entrada e a coluna a ser feita a substituição de valores ausentes, logo em seguida ele oferece 3 técnicas diferentes para a substituição (1-Média, 2-Mediana, 3-Moda), após escolhido o tipo de substituição será gerado um novo arquivo csv com os dados corretamente substituidos.
        - dataset.csv = principal dataset para ser usado como entrada para substituição de dados ausentes no "tarefa2.py"
        - confere.py = arquivo para validação do resultado gerado de "tarefa2.py", dado o nome do arquivo csv de entrada e o numero da coluna desejada, ele irá conferir se existe alguma ocorrencia de '?', que caracteriza um valor ausente naquela coluna. Caso não haja o programa exibe uma mensagem de sucesso, caso contrário ele indica as linhas com valores ausentes.
    . Para execução dos códigos é necessário ter o Python 3 instalado em sua máquina, então rode o comando do python, no nosso caso está configurado para "py" seguido pelo nome do arquivo que deseja executar, exemplos:
        - "py tarefa2.py"
        - "py confere.py"
